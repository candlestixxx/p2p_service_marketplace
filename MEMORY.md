# MEMORY
*   Stack: Next.js 15 (App Router), Prisma, PostgreSQL, TailwindCSS, shadcn/ui.
*   Authentication: NextAuth v5 (Auth.js) using Credentials (bcryptjs) and JWT strategy.
*   Payments: Stripe Connect (Express accounts) handled via Server Actions.
*   Design Preference: Clean, minimalist UI mirroring shadcn default standard, heavy reliance on Server Actions for mutation.
